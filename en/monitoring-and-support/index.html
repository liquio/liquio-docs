    How to monitor a process?

    What are logs?

    BPMN logs display the lifecycle of processes: from schema parsing by the service and message processing to task execution and user interaction. Logs are an important tool for monitoring, analyzing, and optimizing the work of each service, ensuring transparency and control at every stage of the process.

    Process monitoring is performed on the Journals/Process logs tab of the Admin Panel.

    Logs

    Steps:

    To view process logs, follow these steps:

    1. Go to the Journals/Process logs section in the Admin Panel.

    2. Find and click on the process for which you need to view logs.

    3. Review the displayed logs of the selected process.

    All available filters can be combined with each other.
